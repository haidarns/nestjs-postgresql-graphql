# NestJS - PostgreSQL - GraphQL Starter Kit

## Description

A starter kit for NestJS development with PostgreSQL & GraphQL enabled.

## Installation

```bash
$ npm install
```

## Running the app

> For local development, start postgresql container first : `docker-compose up -d`

```bash
# local
$ npm run local

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
